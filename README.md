# calabash_aula_1

Comandos básicos git:

```ruby
git pull - traz a branch/conteúdo de algum lugar para outro
git pull origin master

git clone - Clona o repositório:
git clone https://github.com/cutovas/calabash_aula_1.git

git checkout -b "NOME-Branch" - Cria uma nova branch:
git checkout -b "GC-Autla-01"

git status - verifica o moemnto atual da sua branch:
git status

git add + arquivo - adiciona oarquivo a memória (antes de subir):
git add /workspace/calabash_aula_1/calabash_aula_1/arquivo.rb
git reset HEAD + arquivo

git commit -m "Mensagem" - faz commit (coloque a mensagem):
git commit -m "Este é um exemplo dos comandos básicos do git."

git push - Empurra para o repositório:
git push
```
